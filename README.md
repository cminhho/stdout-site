# stdout — Official Website

Landing site for [stdout](https://github.com/cminhho/stdout): **your standard output for dev tools** — 60+ tools for formatting, converting, encoding & generating. No backend, no sign-up; everything runs locally.

**Live:** [cminhho.github.io/stdout-site](https://cminhho.github.io/stdout-site/) · **App:** [stdout-tools.web.app](https://stdout-tools.web.app/) · **Repo:** [cminhho/stdout](https://github.com/cminhho/stdout)

## Setup

Single-file site. No build step. Just `index.html` served from the repo root.

## GitHub Pages

1. **Settings** → **Pages**
2. **Source:** Deploy from a **branch**
3. **Branch:** `main` · **Folder:** `/ (root)`
4. Save

Site will be available at `https://cminhho.github.io/stdout-site/`.
